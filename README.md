# Pill Reminder System

A simple web-based pill reminder application that helps users receive medication reminders through browser notifications.

## Features
- Browser notifications
- Reminder alerts
- Responsive design
- PWA support

## Technologies Used
- HTML
- CSS
- JavaScript
- Service Worker
- Netlify
